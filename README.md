- **Author:** Daniel J. Umpierrez
- **Version:** 0.1.1
- **License:** UNLICENSE

## Description

File de-duper tool based on xxhash algorithm for faster file comparision.

## Install

```bash
pip install git+https://github.com/havocesp/xxdeduper
```

## Usage

```bash
xxdeduper PATH [PATH [..PATH]]
```
